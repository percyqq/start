From the official Ubuntu PPA
You can install the latest stable version of Redis from the redislabs/redis package repository. Add the repository to the apt index,
update it and install:



$ sudo add-apt-repository ppa:redislabs/redis
$ sudo apt-get update
查看有哪些版本
$ apt-cache madison redis-server
$ sudo apt-get install redis
